Defining the REST API

I. The Clueless's clients will require the services bellow:

  1. Create a player by name (or ID)
  2. Delete a player by name (or ID)
  3. Update a player location by {name and location}
  4. Get the next validation location based on the player current location {name and location}

II. Based on these requirements, the server will provide REST APIs for the services:

  1. /api/player/{name}
    Returns new player name and init location
  2. /api/player/1
    Delete player identified by id (employee 1 in this case)
  3. /api/player/1/update
    Update player location
  4. /api/player/1/nextvalidposition
    Returns all validation location (employee 1 in this case)
